about=[EN] Universal XY Converter is a powerful QGIS plugin for coordinate conversion between any Coordinate Reference Systems (CRS) available in QGIS, with native support for BFTM (Burkina Faso Transverse Mercator). Features: single conversion, batch CSV processing, interactive map picking, vector layer export. [FR] Extension universelle de conversion de coordonnées permettant la conversion entre tous les systèmes de coordonnées disponibles dans QGIS, avec support spécial du BFTM (Burkina Faso Transverse Mercator). Fonctionnalités: conversion simple, traitement batch CSV, sélection interactive sur carte, export direct de couches vectorielles.
